# Stitch Design Animenia

A simple, modern Todo application with a clean UI/UX design.

## Features

- Add, complete, edit, and delete tasks
- Persistent storage using localStorage
- Responsive design
- Dark gradient theme
- Double-click to edit tasks
- Visual feedback on interactions

## How to Run

1. Clone or download this repository.
2. Open `public/index.html` in your web browser.
3. Start adding tasks!

## Design Decisions

- **Color Scheme**: Gradient from `#6a11cb` to `#2575fc` for a vibrant, modern feel.
- **Typography**: Clean, readable sans-serif stack.
- **Interactions**: Hover effects, transitions, and double-click editing for a polished UX.
- **Accessibility**: Proper ARIA labels and semantic HTML.

## Customization

- Edit `src/styles/style.css` to change colors, fonts, or spacing.
- Modify `src/main.js` to adjust behavior or add features.
- Replace SVG assets in `assets/` with your own branding.

## Browser Support

Works in all modern browsers (Chrome, Firefox, Safari, Edge).
